# Custom Projects Plugin

Custom Projects Plugin is a WordPress plugin that allows you to create and manage projects with : 
- title 
- content 
- categories
- image
- date

and a shortcode to display them on your site.

## Features

- Custom Projects custom post type
- Category taxonomy for organizing projects
- [project_list] shortcode for displaying projects

## Installation

1. Upload the `custom-products-management-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

## Usage

1. Create and manage your projects using the "Projects" custom post type.
2. Categorize your projects using the "Categories" taxonomy.
3. Use the `[project_list]` shortcode to display a list of projects on any page or post.

## Note
For custom post type to work, you should set the permalink settings to "Post name" (Settings -> Permalink)
